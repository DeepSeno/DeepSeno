### 中版（300 字，适合媒体报道正文）

> DeepSeno 是一款完全开源的本地 AI 第二大脑。在传统的笔记工具中，用户需要手动记录、整理和检索信息；而在使用云端 AI 服务时，最私密的对话数据不得不交给远程服务器处理。DeepSeno 提供了一种全新的路径：让 AI 自动完成捕获、理解和记忆，同时保证一切都在本地发生。
>
> 在技术层面，DeepSeno 集成了完整的端侧 AI 管线：SenseVoice 语音识别支持中英日韩四语，pyannote + 3dspeaker 实现说话人分离和声纹识别，Silero VAD 做语音活动检测——全部通过 ONNX 在进程内运行，无需 Python 环境。大语言模型能力由本地 llama.cpp 提供（Qwen2.5-14B），实现文本优化、信息提取和 RAG 问答。
>
> 在功能层面，DeepSeno 不只是一个转写工具。它具备 Agent 记忆系统、人物关系图谱、多通道消息集成（飞书/Telegram/企业微信）、MCP 插件市场、定时任务、文档导出（PDF/DOCX/PPTX/Markdown）等完整能力，是一个可扩展的本地 AI 知识平台。
>
> 项目采用 Apache 2.0 许可证，支持 macOS 和 Windows，代码量超过 65,000 行，经过 700+ 次迭代提交。

>
> DeepSeno 是一款完全开源的桌面应用，旨在成为用户的本地 AI 第二大脑。它的核心理念很简单：你的所有语音对话、会议内容、播客笔记、工作沟通，都应该被自动捕获、智能理解、结构化存储，并在你需要时随时可查——而这一切，不需要任何一个字节离开你的电脑。
>
> DeepSeno 在技术上的独特之处在于它构建了一条完整的端侧 AI 管线。语音识别采用 SenseVoice（支持中英日韩），说话人分离使用 pyannote + 3dspeaker，语音活动检测基于 Silero VAD，全部通过 sherpa-onnx 的 Node.js 原生绑定在 Electron 进程内运行，模型总大小仅约 263MB。大语言模型能力由本地 llama.cpp 提供，默认使用 Qwen2.5-14B 处理文本优化、信息提取、摘要生成和 RAG 问答。整套方案不依赖 Python 环境，不需要 HuggingFace token，开箱即用。

> 项目基于 Electron + React + TypeScript 构建，使用 Node.js 24 原生 SQLite 做结构化存储（WAL 模式 + FTS5 全文搜索），sqlite-vec 做向量语义搜索，代码量超过 65,000 行，经过 700+ 次迭代提交。采用 Apache 2.0 许可证，支持 macOS（Apple Silicon + Intel）和 Windows。
>
> 在云端 AI 成为主流的今天，DeepSeno 代表了另一种可能：AI 能力足够强大，但你的数据不需要离开你的设备。

## 二、核心卖点（媒体引用要点）

### 1. 完全本地运行
所有 AI 处理在设备端完成。语音识别、说话人分离、声纹识别通过 ONNX 在进程内运行；大语言模型通过本地 llama.cpp 运行。没有任何数据上传到云端，即使断网也能正常工作。

### 2. 不只是转写，是第二大脑
自动从语音中提取待办事项、会议纪要、决策、联系人。三层记忆架构让 AI 真正"记住"你的信息。RAG 问答引擎让你可以用自然语言查询所有历史内容。

### 3. 开箱即用，无需配置环境
不需要安装 Python，不需要 HuggingFace token，不需要配置 GPU 驱动。下载安装包，首次引导完成后即可使用。ONNX 模型仅约 263MB（对比 PyTorch 模型 3GB+）。

### 4. 可扩展的本地 AI 平台
支持 MCP 插件协议、多通道消息集成、定时任务调度、局域网服务。不是一个封闭工具，而是一个开放的本地 AI 知识平台。

### 5. 真正的开源
Apache 2.0 许可证，完整源代码，无付费墙，无功能阉割。社区可以自由使用、修改、分发。

## 四、技术亮点（适合技术媒体）

### 端侧 AI 管线
- 从 Python 子进程迁移到 sherpa-onnx-node 原生绑定，实现全 JavaScript 技术栈
- 冷启动从 60 秒降至毫秒级
- 移除 Python 依赖，安装包减少 1.1GB+
- ONNX 模型 ~263MB vs PyTorch 模型 3GB+

### 数据库架构
- Node.js 24 原生 SQLite WAL 模式实现高并发读写
- FTS5 全文搜索索引实现毫秒级文本检索
- sqlite-vec 向量扩展实现语义搜索，无需外部向量数据库

### 插件生态
- 基于 Anthropic MCP（Model Context Protocol）协议
- 插件可以是纯指令注入（System Prompt）或完整工具调用
- 插件市场支持一键安装

### Agent 架构
- 三层记忆系统：短期/长期/语义记忆
- 自动事实提取与冲突检测（余弦相似度 > 0.85）
- 主动洞察引擎：不只回答问题，还能主动发现关联
